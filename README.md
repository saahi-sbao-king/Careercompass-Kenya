# CareerCompass Kenya - Project Blueprint

CareerCompass Kenya is a high-fidelity career guidance platform custom-built for students at Frere Town Secondary School and the wider Kenyan Competency-Based Education (CBE) system.

## 🚀 Core Features

### 1. AI-Powered Multiple Intelligence Assessment
- **45-Question Quiz**: Evaluates 9 intelligence modalities (Linguistic, Logical-Mathematical, Spatial, Bodily-Kinesthetic, Musical, Interpersonal, Intrapersonal, Naturalist, and Existential).
- **Pathway Classification**: Maps MI profiles to the 3 official Senior School Pathways: **STEM**, **Arts & Sports Science**, or **Social Sciences**.
- **O(n) Scoring**: Optimized algorithm to ensure zero memory bottlenecks during processing.

### 2. Hobby-Career Prediction Engine
- **Hobby-Career Analyst**: Maps unique student hobbies (e.g., beekeeping, calligraphy, coding, mock trials) to specialized professional personas.
- **Market Intelligence**: Provides realistic Kenyan salary ranges (KES), market demand scores (1-10), and official CBE subject alignments.

### 3. Subject Combination & Cluster Tools
- **Pathfinder AI v3.0**: Analyzes specialized 3-elective mixes to predict professional outcomes and university course alignment.
- **KUCCPS Calculator**: Estimations for Cluster Weight points based on official formulas.
- **Official Catalogue**: Digital library of all Senior School subject combinations and tracks.

### 4. Personalization & AI Coaching
- **AI Study Coach**: Generates dynamic 7-day weekly timetables that balance academic subjects with personal interests and learning styles.
- **AI Avatar Generator**: Uses Imagen 3.0 to create professional-persona portraits for student profiles.
- **Career Storyteller**: AI-generated inspiring stories about Kenyan success in specialized fields.

### 5. Professional Reporting
- **PDF Report Generator**: High-fidelity export of career blueprints, including MI charts, pathway analysis, and elective recommendations for school records.

### 6. Administration & Mentorship
- **Admin Console**: Analytics for school counselors to monitor pathway distribution and student progress.
- **Mentorship Hub**: Direct links to Kenyan mentorship programs and official resources (KUCCPS, KICD).

## 🛠 Technical Specification

- **Framework**: Next.js 15 (App Router), React 19.
- **Styling**: Tailwind CSS with ShadCN UI components.
- **Backend**: Firebase Suite (Auth, Firestore, Storage, Analytics).
- **Resource Config**: 1GiB RAM / 1 CPU configuration for stable AI execution.
- **AI Engine**: Google Genkit using Gemini 1.5 Flash & Imagen 3.0.
- **Security**: reCAPTCHA Enterprise, Firestore Security Rules (Path-based ownership).

## 🇰🇪 CBE Standards Compliance
- **11-Subject Rule**: Enforces the 4 Compulsory Subjects + 7 Electives structure.
- **Specialization Rule**: Ensures alignment with the 3-elective specialized track requirement for pathway cluster analysis.
- **Terminology**: Uses official MoE/KICD naming (e.g., "Core/Essential Mathematics", "Community Service Learning").
