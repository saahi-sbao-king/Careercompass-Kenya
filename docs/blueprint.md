# **App Name**: CareerCompass Kenya

## Core Features:

- User Authentication: Secure user login/signup with email/password, Google, and phone number; supports 'Student' and 'Teacher' roles with admin privileges managed through Firestore.
- Career Assessment Quiz: A comprehensive multi-step quiz based on Howard Gardner's Multiple Intelligences theory, including saving progress to localStorage.
- Personalized Recommendations: Display detailed results based on MI scores, showing pathway, confidence score, top intelligences, suggested careers/subjects; allows PDF download and saving results to Firestore.
- User Dashboard: Personalized dashboard displaying assessment results, saved career interests, and a study timetable. It incorporates an AI Study Coach that provides personalized study recommendations.
- Career Exploration: Interactive career exploration grid with details, user saving functionality to Firestore, and an AI-powered 'Personalized Suggestions' feature based on interests.
- AI-Powered Story Generator: Generates a short story based on user prompts.
- AI Avatar Generator: Users can create a custom AI avatar by entering prompts using Imagen.
- Subject Combination Explorer: Get suggestions for careers based on subjects excelled.
- Admin Dashboard: Admin interface with summary statistics, user management, and Recharts visualization of dominant intelligences distribution from Firestore. Access managed via Firestore roles.

## Style Guidelines:

- Primary color: Indigo-blue (#4A77D4), reflecting trust and guidance.
- Background color: Light gray (#F0F2F5), offering a clean and modern interface.
- Accent color: Blue-violet (#7952B3), for emphasis and calls to action.
- Body and headline font: 'Inter', sans-serif, for a clean, contemporary feel.
- Lucide React icons for a consistent, modern look.
- Responsive layout using Tailwind CSS and ShadCN UI components to ensure a seamless experience across devices.
- Subtle transitions and animations to enhance user engagement.
